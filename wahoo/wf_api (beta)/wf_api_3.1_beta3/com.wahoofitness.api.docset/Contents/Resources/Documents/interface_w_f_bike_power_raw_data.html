Detailed Description

Represents the raw data available from the ANT+ Bike Power sensor.

ANT+ sensors send data in multiple packets. The WFBikePowerRawData combines the more advanced (less commonly used) of this data into a single entity. The data represents the latest of each data type sent from the sensor.
