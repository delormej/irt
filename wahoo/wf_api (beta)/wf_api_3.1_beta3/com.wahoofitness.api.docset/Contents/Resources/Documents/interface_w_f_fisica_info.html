Detailed Description

Represents information describing the properties and capabilities of the currently connected fisica device.

Property Documentation

- (ULONG) firmwareVersion [read, write, assign]

Gets the fisica device firmware version.

The firmware version is encoded in the lower three bytes:
B2: The major version number.
B1: The minor version number.
B0: The revision number.

- (UCHAR) maxAntConnections [read, write, assign]

Gets the maximum number of ANT devices which may be connected at one time.

- (NSString*) model [read, write, retain]

Gets the fisica device model name.

- (ULONG) serialNumber [read, write, assign]

Gets the fisica device serial number.

- (BOOL) supportsAntFSClient [read, write, assign]

TRUE if the fisica hardware supports ANT FS Client mode, otherwise FALSE.

- (BOOL) supportsFitnessEquipment [read, write, assign]

TRUE if the fisica hardware supports Fit1e enabled Fitness Equipment, otherwise FALSE.

- (BOOL) supportsProximitySearch [read, write, assign]

TRUE if the fisica hardware supports proximity search, otherwise FALSE.
