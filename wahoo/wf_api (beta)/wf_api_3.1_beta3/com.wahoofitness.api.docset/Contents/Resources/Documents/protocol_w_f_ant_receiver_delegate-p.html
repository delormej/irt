Detailed Description

The WFAntReceiverDelegate protocol declares the interface for an ANT message receiver used when the WFHardwareConnector is in advanced mode.

The WFAntReceiverDelegate protocol should be adopted by classes which will receive ANT messages directly from the WFHardwareConnector. A reference to such a delegate is passed to the WFHardwareConnector via the initializeAdvancedMode: (WFHardwareConnector) method. The WFHardwareConnector will then forward all ANT Messages it receives to the delegate.
